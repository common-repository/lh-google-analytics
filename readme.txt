Add one or more Google Analytics trackers to your website, unobtrusively

== Description ==

A light weight and unobtrustive [Google Analytics](http://www.google.com/analytics/) plugin that lets you add one or more trackers.

Features are:

* Handles multiple tracking codes
* Script loading is deferred
* Is 100% unobtrusive javascript
* Handles handles userID tracking (you need to enable this also within the google analytics dashboard)


== Installation ==

1. Move the 'lh-google-analytics' folder to the `/wp-content/plugins/` directory OR search for 'LH Google Analytics' and add the plugin using WordPress plugin browser.
2. Activate the plugin through the 'Plugins' page in WordPress
3. Add one or more web property IDs (UA-XXXXXXX-X strings) on the settings page, seperating each tracker by a comma
